# 1. Console Shortcuts
---

`'Cmd + Shift + C'` : 콘솔로 포커 옮기기

`'Cmd + K'` : 콘솔 Clear

`'Cmd + \'` : 현재 라인 브레이크 포인트 추가/삭제

`'Cmd + Y'` : 브레이크 포인트 활성/비활성


# 2. Changing debug steps
---

`'n'` : step over, 다음 라인 실행

`'s'` : step in

`'fin'` : step out

`'c'` : continue, 다음 브레이크로 이동
