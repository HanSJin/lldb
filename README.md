# 1. 변수의 값, 상태 출력
---

Commands: `expression`, `e`, `print`, `po`, `p`

`(lldb) expression ...`

`(lldb) po ...`

`(lldb) e ...`

# 1. Console Shortcuts
---

`'Cmd + Shift + C'` : 콘솔로 포커 옮기기

`'Cmd + K'` : 콘솔 Clear

# 2. Changing debug steps
---

`'r'` : run, Run the process
`'n'` : step over, 다음 라인 실행
`'s'` : step in, 다음 라인 실행
`'fin'` : step out, 다음 라인 실행
`'c'` : continue, 다음 Br로 이동
`'exit'` : 디버깅 종료
